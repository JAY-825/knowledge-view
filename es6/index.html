# ES6

ES6， 全称 ECMAScript 6.0 ，2015年6月正式发版，成为国际标准。

# 什么是ES6

ES6是ECMAscript标准中变动最大的一个版本，新增了许多新的语法特性。

这也让js代码能够适应更复杂的应用，实现代码库之间的共享，不断迭代维护新版本。

# ES6新增特性

  • let、const
  • 数组扩展
  • 对象扩展
  • 函数扩展
  • Set、Map
  • Promise
  • Generator
  • Proxy
  • Module
  • Decorator
